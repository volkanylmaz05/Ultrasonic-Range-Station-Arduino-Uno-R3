# Distance Alarm Project (Arduino)

This project is a simple distance alarm. It uses an HC-SR04 ultrasonic sensor. 
If an object comes closer than 20cm, the LED and Buzzer will turn on.
-----------------------------------------------------------------
## Components
* Arduino Uno
* HC-SR04 Sensor
* Buzzer
* Red LED
* 220 Ohm Resistor
----------------------------------------------------------------
## Pins
* Trig: Pin 10
* Echo: Pin 9
* Buzzer: Pin 12
* LED: Pin 5
----------------------------------------------------------------
## How to Setup
1. Connect the parts like the circuit image.
2. Open the code in PlatformIO or Arduino IDE.
3. Download the NewPing library.
4. Upload the code and open the Serial Monitor (9600 baud).
